📊 Modelo de Datos de Navegación

El encabezado contiene enlaces clave para facilitar la navegación del usuario.

  • Enlace a Contacto: Enlace directo a la página de contacto para asistencia al cliente.
  • Selector de Idioma: Permite a los clientes elegir el idioma de visualización de la tienda.
  • Selector de Moneda: Opción para ver los precios en diferentes monedas.
  • Enlace de Autenticación: Lleva a la página de inicio de sesión. Cambia a "Cerrar sesión" una vez el usuario está autenticado.
  • Carrito de Compras: Muestra el número de artículos en el carrito y da acceso al contenido del carrito.
